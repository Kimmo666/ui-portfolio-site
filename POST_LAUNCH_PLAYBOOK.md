# Post-Launch Optimization Playbook (First 2 Weeks)

## Goals

- Identify top-performing case studies.
- Improve contact conversion from portfolio traffic.
- Strengthen decision rationale and measurable outcomes in case content.

## Metrics To Track

- Page views per case page
- Avg. engaged time per case page
- Click-through rate to `contact.html`
- Contact form submissions (or mailto clicks)

## Weekly Workflow

### Week 1

- Add analytics script (Plausible or Google Analytics) to all pages.
- Establish baseline:
  - Most visited case page
  - Lowest dwell time page
  - Contact click conversion by source page

### Week 2

- Reorder case cards in `index.html` and `works.html` by performance.
- Expand weakest case page:
  - Add clearer problem statement
  - Add stronger visual process explanation
  - Add explicit business outcome
- Improve CTA copy if contact conversion is below 2%.

## Reorder Rule (Simple)

Sort by:

1. Contact clicks generated
2. Avg. engaged time
3. Page views

## Content Upgrade Checklist

- Did the case explain the business problem in the first paragraph?
- Did it show design decisions, not only final screens?
- Did it quantify outcomes with numbers?
- Did it include one clear CTA to contact?
