# Works Image Mapping

Drop your real portfolio images under this directory, and the site will auto-load them.

## Folder Structure

- `assets/works/finflow/`
- `assets/works/shoplane/`
- `assets/works/healthbridge/`

## Naming Convention

### Home Page

- `finflow/home-cover.jpg`
- `finflow/home-system.jpg`
- `shoplane/home-cover.jpg`
- `shoplane/home-detail.jpg`
- `healthbridge/home-cover.jpg`

## Notes

- Default extension is `.jpg`.
- If a local file is missing, the site automatically falls back to the current placeholder image URL.
- You can switch to `.png` for a specific image by adding `data-ext="png"` on that `<img>` tag.
